# ✅ E-Commerce Testing Project

- This repository contains an automated testing suite for an e-commerce web application. It follows the **Page Object Model (POM)** design pattern and includes functional, integration, and end-to-end tests.
---
## 🔍 Overview

- This project is designed to automate testing of key e-commerce functionalities such as authentication, product browsing, cart management, and checkout.

- The framework is structured using **Page Object Model (POM)** to ensure scalability, maintainability, and clean separation between test logic and UI elements.

---

## 🚀 Features

* Automated UI testing for core e-commerce flows
* Page Object Model (POM) implementation
* End-to-End (E2E) testing scenarios
* Reusable test components
* Scalable and maintainable test structure

---

## 📁 Structure

* Pages: UI page classes (Cart, Checkout, Login, etc.)
* Tests: Functional & E2E test cases

---
 
## 🔄 Test Scenarios

* User Registration
* User Login & Logout
* Product Search
* Add to Cart
* Wishlist Management
* Checkout Process
* Payment Flow
* End-to-End Purchase Flow
